# fe-marketplace

Risidio Auction client application. Features include:

- marketplace - bid, buy, sell
- timed auctions
- live auctions
- crowdfunding

## Project setup

```bash
nvm use 12.16.3
```

or later

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Run your unit tests

```bash
npm run test:unit
```

### Run your end-to-end test

```bash
npm run test:e2e
```

### Lints and fixes file

```bash
npm run lint
```
